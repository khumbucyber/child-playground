# なかまわけ名人 ～どめいんもどらー～ 仕様書

作成日: 2026-03-11

---

## コンセプト

「分類」は思考の根幹であり、ドメインモデリングの出発点。
**「同じものでも、文脈によって別のグループに属せる」** という体験を5歳が直感として身につけるゲーム。

大人が陥る2つの罠を予防する：
1. 違うものを一緒にしてしまう（議論が噛み合わない原因）
2. 同じものを別と見てしまう（情報過多で立ち往生する原因）

サブタイトル「どめいんもどらー」は遊び心で添える。ちびっこが意味を知らなくても良い。

---

## ゲームフロー

```
タイトル画面
    ↓
ステージ1〜5（順番に進む）
    ↓ （各ステージ）
    問題表示（「うごくものをえらんでね！」）
    → アイテム選択（タップでON/OFF）
    → 「きめた！」ボタン
    → 正解・不正解フィードバック
    → 「ひとことメモ」（文脈またぎの気づき）
    → 次のステージへ
    ↓
クリア画面（どめいんモデル表示）
```

```
🐟 さかな
  うごくもの ✓
  たべもの  ✓
  いきもの  ✓

🍎 りんご
  たべもの  ✓
  まるいもの ✓

🚂 でんしゃ
  うごくもの ✓
  のりもの  ✓
```

## UI・UX要件

- スマホ縦持ち最優先（`100dvh`、`safe-area-inset`対応）
- アイテムカード：最低72px角、タップで選択状態トグル
- 選択中：明るいボーダー＋拡大アニメーション
- 正解：🎉緑ふわり演出、不正解：❌軽く揺れる演出
- 文字：ひらがな中心、大きめ
- 色：明るいパステル系
- フォント：Hiragino Maru Gothic Pro → BIZ UDPGothic → Kosugi Maru
- `touch-action: manipulation`でダブルタップ防止

## 教育的ねらい（保護者向け補足）

- **属性の抽出**：何に着目して仲間わけするかを考える
- **文脈依存**：「問い」によって正解が変わることを体験
- **多重所属**：同じものが複数のグループに属せることを発見
- **反事実的思考**：「さかなはたべものなのに、うごくものでもあるの？」という驚きと探求
