#### Q3（こくご）【問題文差し替え】

**修正前**
```
text: '「あいうえお」は なんびょうし の もじ？'
options: ['３つ', '４つ', '５つ'], answer: 2
```
**修正後**
```
text: '「あいうえお」は なんこ の もじ？'
options: ['３こ', '４こ', '５こ'], answer: 2
```
**理由**: 「びょうし（拍子）」は音楽用語であり5歳には伝わらない。「なんこ の もじ？」という平易な表現に差し替え。answer は変更なし（5こ＝index 2）。

#### Q12（こくご）【問題文・選択肢差し替え】

**修正前**
```
emoji: '☀️', text: '「あつい」の はんたいの ことばは？'
```

**修正後**

#### Q33（さんすう）【問題文修正】

**修正前**
```
text: 'きんぎょが ６ぴき。３ぴき かうと なんびき？'
options: ['２ひき', '３ひき', '４ひき'], answer: 1
```
**修正後**
```
text: 'きんぎょが ６ぴき。３ぴき にがすと なんびき のこる？'
options: ['２ひき', '３ひき', '４ひき'], answer: 1
```
**理由**: 「かう」は「買う」と「飼う」の両方に読め、5歳が「もらって増える」と誤解する可能性がある。「にがす（逃がす）」に変更して「減る」操作を明確化。answer・選択肢は変更なし。

#### Q44（りか）【問題文・選択肢差し替え】

**修正後**
