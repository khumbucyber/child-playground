# 仕様書 — クイズ (`quiz.html`)

## 概要
5歳向けの三択クイズ。絵文字つきの問題に対して3つの選択肢から答えを選ぶ。
全10問・ランダム出題。6教科から幅広く出題。

## 対象ユーザー
5歳前後。ひらがなは読める。漢字・英語（えいご問題を除く）は使わない。

---

## 画面フロー

```
タイトル画面 → クイズ中 → 結果画面 → タイトルへ戻る
```

### タイトル画面
- キャラクター（🦊）が「もんだいを だすよ！」と吹き出し
- 「はじめる！」ボタン
- 「もどる」ボタン（index.html へ）

### クイズ中
```
┌──────────────────────────────┐
│  もんだい 3 / 10    [もどる]  │
│  ████░░░░░░  進捗バー         │
│  [こくご]  ← 教科バッジ       │
├──────────────────────────────┤
│                              │
│          🐘                  │
│   「ぞうは なにを つかって    │
│      みずを のむ？」          │
│                              │
├──────────────────────────────┤
│  ① くち                      │
│  ② はな（はなをのばして）     │  ← 三択ボタン（縦並び）
│  ③ しっぽ                    │
└──────────────────────────────┘
```

### 正解・不正解フィードバック
- 正解ボタンを緑にハイライト ＋ 「ピンポーン🎉」テキスト ＋ 効果音（Web Audio）
- 不正解ボタンを赤にハイライト → 正解ボタンを緑表示 ＋ 「ざんねん！😢」＋ 効果音
- 1.2秒後に自動で次の問題へ

### 結果画面
- スコアを大きく表示（例: `7 / 10 もんせい！`）
- 満点: 紙吹雪 ＋「まんてん！すごい！🏆」
- 8〜9問: 「すごい！🌟」
- 5〜7問: 「よくできました！😊」
- 〜4問: 「またちょうせん！💪」
- 「もういちど」「もどる」ボタン

## デザイン要件
- 背景: 夜空風（紺→紫グラデーション）に星のアニメ ✨
- 問題文フォントは大きく（20px以上）・ひらがなのみ
- 選択肢ボタンは縦に3つ並べる。高さ56px以上・横幅いっぱい
- ①②③ の番号を左端につけて視覚的に区別しやすくする
- 正解時: ボタンが緑にアニメ、不正解時: 選んだボタンが赤・正解が緑
- 進捗バーは虹色グラデーション
- キャラクター🦊 は画面の隅で正解・不正解にリアクションする
- 教科バッジ（「こくご」「さんすう」など）を問題カードの上部に表示

## 技術制約
- HTML / CSS / Vanilla JS のみ（1ファイル）
- 外部ライブラリ不使用
- `100dvh` + `env(safe-area-inset-bottom)` でスマホ対応
- 効果音は Web Audio API で生成（外部ファイル不使用）
