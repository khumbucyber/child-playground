# 仕様書 — おえかき (`oekaki.html`)

## 概要
指でなぞって自由にお絵かきできるキャンバスアプリ。
保存・印刷はせず、その場で楽しむことに特化する。

## 対象ユーザー
5歳前後の子ども。親がスマホを渡して一人で遊ぶ想定。

---

## 画面構成

```
┌─────────────────────────┐
│  🎨 おえかき   [もどる] │  ← ヘッダー（固定・細め）
├─────────────────────────┤
│                         │
│   キャンバス            │  ← flex: 1（残り全高さ）
│   （白背景）            │
│                         │
├─────────────────────────┤
│  色パレット（横スクロール）│  ← 固定高さ 64px
│  太さ選択  undo  きれいに│  ← 固定高さ 60px
└─────────────────────────┘
```

---

## 機能要件

### キャンバス
- `<canvas>` 要素でキャンバス領域全体を占有
- タッチ（`pointermove`）で描画。マルチタッチは無視（1本指のみ）
- 背景は白
- ページスクロールを抑止（`touch-action: none`）

### 色パレット
- 横スクロール可能な色ボタン列
- 初期色セット（最低12色）:
  `#ff4757` `#ff6b81` `#ffa502` `#ffdd59` `#7bed9f`
  `#2ed573` `#1e90ff` `#5352ed` `#a29bfe` `#fd79a8`
  `#2d3436` `#ffffff`
- 選択中の色はリング強調（`outline`）
- 色ボタンは直径 44px・丸型

### ブラシ太さ
- 細（4px）・中（12px）・太（28px）の3段階
- アイコンは ●（大きさ違い）で視覚的に区別

### アクション
| ボタン | 動作 |
|---|---|
| `↩ もどす` | 1ストローク Undo（最大30ストローク） |
| `🗑 きれいに` | 全消去（確認ダイアログなし）。押したらポップで演出 |
| `もどる` | `index.html` へ戻る |

---

## デザイン要件
- カラフルで明るいUI。ヘッダーは虹グラデーション
- 色パレットのボタンはグラデーション球体風（`radial-gradient`）
- 「きれいに」ボタン押下時に紙吹雪アニメを短く再生（0.8秒）
- フォント: `Hiragino Maru Gothic Pro` 系

## 技術制約
- HTML / CSS / Vanilla JS のみ（1ファイル）
- 外部ライブラリ不使用
- `100dvh` + `env(safe-area-inset-bottom)` でスマホ対応
