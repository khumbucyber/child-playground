# 仕様書 — しゃしんゲーム (`shashin.html`)

## 概要
画面上を動く被写体（電車・ちょうちょ・わんちゃんなど）が
ファインダー枠の中に入った瞬間にタップして「写真を撮る」タイミングゲーム。
被写体は動き続け、枠から外れたり画面外へ飛び出したりする。
うまく撮れると「いい写真」が記録され、ステージクリアになる。

## 対象ユーザー
5歳前後。直感的なタイミング操作で楽しめる。

---

## 画面フロー

```
タイトル画面 → ステージ選択 → ゲーム中 → 撮影結果 → 次のステージ or リザルト
```

---

## タイトル画面

- タイトル「📷 しゃしん ゲーム」
- カメラのシャッター音風アニメ（画面全体がフラッシュ）
- 「はじめる！」ボタン
- 「もどる」ボタン（index.html へ）

---

## ステージ構成

全3ステージ。クリア条件は「各ステージの被写体を規定枚数撮影すること」。

| ステージ | 被写体 | 登場数 | 必要撮影数 | 特徴 |
|---|---|---|---|---|
| 1 | 🐶 わんちゃん | 5匹 | 3匹 | 直線移動・比較的ゆっくり |
| 2 | 🦋 ちょうちょ | 6匹 | 4匹 | 波打ち移動・中速 |
| 3 | 🚂 でんしゃ | 4本 | 3本 | 高速直線・短時間で通過 |

## ゲームロジック

### 被写体の動き

- 被写体は画面の**左端または右端の画面外**からランダムに出現する
- 出現後、直線または曲線（サイン波）を描いて移動し、**反対側の画面外へ消える**
- 消えたら次の被写体が少し間をおいて出現（インターバル 1〜2秒）
- 1匹ずつ登場（複数同時は出さない）

### 移動パターン（被写体別）

| 被写体 | 移動パターン | 速度 |
|---|---|---|
| 🐶 わんちゃん | 横方向の直線（Y座標は出現時にランダム固定） | 遅い（画面横断 3.5秒） |
| 🦋 ちょうちょ | 横方向 ＋ サイン波で上下（振幅 60px・周期 1.5秒） | 中速（画面横断 2.5秒） |
| 🚂 でんしゃ | 横方向の直線（Y座標固定・中央付近） | 速い（画面横断 1.8秒） |

### ファインダー枠

- 画面中央やや上に**固定**で表示（プレイヤーは動かせない）
- サイズ: 画面幅の約 35%・正方形
- 枠のスタイル: 角に L字ブラケット（カメラのファインダー風）、背景は半透明
- 被写体が枠内に入ったとき、枠が**緑色に光る**（視覚フィードバック）

### 判定ロジック（シャッターボタンタップ時）

被写体の中心座標がファインダー枠の内側にあるかどうかで判定。

| 状態 | 結果 |
|---|---|
| 被写体の中心が枠の内側 | ✅ **撮影成功** |
| 被写体の中心が枠の外 | ❌ **撮影ミス**（シャッター音のみ） |
| 被写体が画面外（不在） | ❌ 空振り |

### 撮影成功演出
1. 画面全体が白くフラッシュ（0.15秒）
2. 「カシャッ！📸」テキストがポップ
3. 撮影した被写体がその場で止まり、**ポラロイド写真風のカード**に変わるアニメ（0.8秒）
4. カードが上部の「撮影済み」エリアに飛んでいく
5. 必要枚数達成でステージクリア

### 撮影ミス演出
- 「カシャ…😅」テキストが薄く出て消える（0.5秒）
- 被写体はそのまま動き続ける

---

## 撮影結果（ステージクリア時）

- 撮影に成功した被写体をポラロイド風カードで並べて表示
- 各カードに「いい しゃしん！ 🌟」などコメント
- 「つぎへ！」ボタン（次ステージ）または「おわり」ボタン（リザルトへ）

---

## リザルト画面（全3ステージ終了後）

- 撮影できた合計枚数と「アルバム」風に写真を並べて表示
- 全クリア（全ステージ必要数達成）: 紙吹雪 ＋「さいこうの カメラマン！ 🏆」
- 部分クリア: 「よく とれたね！📷」
- 「もういちど」「もどる」ボタン

---

## デザイン要件

- 背景: 公園・空をイメージした薄い青空グラデーション
- ファインダー枠: 黒または濃い緑のブラケット、被写体が入ると緑に光る
- シャッターボタン: 大きな円形（直径 80px 以上）、カメラのシャッター風のグレー〜白
- ポラロイドカード: 白背景・厚みのある影・下部に薄いクリーム色の余白
- 被写体の絵文字サイズ: 48〜56px
- フォント: `Hiragino Maru Gothic Pro` 系、ひらがなのみ

## サウンド（Web Audio API で生成・外部ファイル不使用）

| イベント | 音 |
|---|---|
| シャッター成功 | 高めのクリック音 ＋ 短い上昇音 |
| シャッターミス | 低いクリック音のみ |
| ステージクリア | 明るい上昇メロディ |
| 全クリア | ファンファーレ風 |

---

## 技術制約

- HTML / CSS / Vanilla JS のみ（1ファイル完結）
- 外部ライブラリ不使用
- アニメーションは `requestAnimationFrame` ループで管理
- 被写体位置は座標で管理（CSS `transform: translate` で描画）
- `100dvh` + `env(safe-area-inset-bottom)` でスマホ対応
- タップ遅延を排除（`touch-action: manipulation`）
