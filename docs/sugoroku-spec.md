# すごろくあそび 仕様書

**バージョン**: 1.0
**作成日**: 2026-03-28
**ファイル**: `public/sugoroku.html`

---

## 概要

5歳前後の子ども向けすごろくゲーム。知育ミニゲーム付きのマスを含み、対人・CPU対戦に対応。

---

## モード

| モード | 説明 |
|---|---|
| ひとりで あそぶ | CPU対戦（プレイヤー1人 vs CPU1体） |
| ふたりで あそぶ | 同端末で交互プレイ（プレイヤー2人） |

---

## キャラクター

4種から選択（色違いのコマで区別）：
- 🐶 いぬ
- 🐱 ねこ
- 🐰 うさぎ
- 🐻 くま

CPU は選ばれなかったキャラクターから自動選択。

---

## ゲームボード

- **マス数**: 25マス（スタート + 23マス + ゴール）
- **形状**: 蛇行型（スマホ縦持ちに最適化）
- **1プレイ目安**: 5〜8分

### マス種別

| 種別 | 割合 | 内容 |
|---|---|---|
| 🟡 ふつう | 40% | 何も起きない |
| ⭐ ラッキー | 15% | 2〜3マス進む |
| 💨 アクシデント | 15% | 1〜2マス戻る |
| 🔄 やすみ | 10% | 1回休み |
| 🌀 ワープ | 5% | 指定マスへ移動 |
| 🎯 チャレンジ | 15% | 知育ミニゲーム（正解で+1マスボーナス） |

---

## 知育ミニゲーム

難易度はゲーム開始時に選択（Lv1〜3）。正解すると+1マスボーナス。不正解でもゲームは止まらない（ペナルティなし）。

### カテゴリ

| カテゴリ | 内容例 |
|---|---|
| 🔢 かずあそび | 絵の個数を数える / 足し算（1〜10） |
| 🔤 ことば | しりとり / 「〇から始まる言葉」選択 |
| 🎨 いろ・かたち | 同じ色を選ぶ / 形の名前を当てる |
| 🤔 なぞなぞ | ひらがなで読めるなぞなぞ |
| 👀 きおく | カードを見て → 隠す → 何だった？ |

### 難易度

| Lv | 選択肢 | 制限時間 |
|---|---|---|
| 1 | 3択 | なし |
| 2 | 4択 | なし |
| 3 | 4択 | 10秒 |

---

## CPU の動作

- サイコロ目はランダム（プレイヤーと同条件）
- チャレンジマス：Lv1は80%正解、Lv2は60%正解、Lv3は40%正解（疑似的に）
- ターン間に0.8秒の「考え中」演出

---

## UX・設計原則

- 即時フィードバック（正誤を0.5秒以内に音＋エフェクトで表示）
- 不正解時は「おしい！」でポジティブに励ます
- ひらがなのみの問題文
- タップ操作のみで完結
- `touch-action: manipulation` でダブルタップズーム無効
- `env(safe-area-inset-bottom)` でiPhoneセーフエリア対応

---

## 技術

- HTML / CSS / Vanilla JS（すべてインライン）
- ボードはDOMで描画（CSS Grid）
- コマ移動はCSSアニメーション
- サイコロは3Dアニメーション風
