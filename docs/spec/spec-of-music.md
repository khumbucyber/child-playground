## ゲームデザイン

### 基本ルール

- 音符が画面上から落ちてくる
- 画面下部の「判定ライン」に近づいたタイミングでタップ
- タイミングに応じて PERFECT / GOOD / MISS の判定が出る
- 曲が終わったらスコア（★の数など）を表示

### 判定基準（目安）

| 判定 | タイミング誤差 |
|---|---|
| PERFECT | ±80ms 以内 |
| GOOD | ±180ms 以内 |
| MISS | それ以外（またはタップしない） |

### スコア

- PERFECT = 10点、GOOD = 5点、MISS = 0点
- 曲終了後に合計点数と★評価（3段階）を表示

---

## レーン構成

- レーン数：**4本**（スマホ縦持ちで指が届きやすい幅）
- 各レーンに固定の音（ド・レ・ミ・ファ など）を割り当て
- レーンの色はカラフルに（赤・黄・緑・青 など）
- レーン幅はスマホ画面幅を均等分割

```
[ ド ][ レ ][ ミ ][ ファ ]   ← タップボタン（画面下部）
```

### 使用楽曲（すべてパブリックドメイン）

#### 1. きらきらぼし（Twinkle Twinkle Little Star）

```
ド ド ソ ソ ラ ラ ソ　（ファ ファ ミ ミ レ レ ド）
ソ ソ ファ ファ ミ ミ レ　（ソ ソ ファ ファ ミ ミ レ）
ド ド ソ ソ ラ ラ ソ　（ファ ファ ミ ミ レ レ ド）
```

#### 2. かえるのうた

```
ド レ ミ ファ ミ レ ド
ミ ファ ソ ラ ソ ファ ミ
ド ド ド ド　レ レ レ レ
ミ ミ ミ ミ　ド ド ド ド
```

#### 3. メリーさんのひつじ（Mary Had a Little Lamb）

```
ミ レ ド レ ミ ミ ミ
レ レ レ　ミ ソ ソ
ミ レ ド レ ミ ミ ミ
レ レ ミ レ ド
```

### 音符データ形式

```js
// 各音符：{ lane: 0-3, time: 秒, note: 'C4' など }

      { lane: 0, time: 0.6,  note: 'C4' }, // ド
      { lane: 3, time: 1.2,  note: 'G4' }, // ソ
      // ...
    ]
  },
  kaeru: { ... },
  mary:  { ... }
}
```

## 音声生成（Web Audio API）

- `OscillatorNode` を使って音を生成（外部音声ファイル不要）
- 波形タイプ: `sine`（やわらかい音色）
- 各ノートの周波数マッピング：

```js
const FREQ = {
  'C4': 261.63, 'D4': 293.66, 'E4': 329.63,
  'F4': 349.23, 'G4': 392.00, 'A4': 440.00,
  'B4': 493.88, 'C5': 523.25
};
```

- タップ時に対応する音を鳴らす（フィードバック音）
- MISS 時は音なし（または短い低音）

## 画面構成

```
┌────────────────────────────┐
│  🎹 おんがく     [もどる]  │  ← ヘッダー
├────────────────────────────┤
│  曲名：きらきらぼし        │
│  スコア：0   コンボ：0     │
├────────────────────────────┤
│                            │
│   ↓  ↓  ↓  ↓              │  ← Canvas（音符が落ちる）
│                            │
│  ━━━━━━━━━━━━━━━━━━━━━     │  ← 判定ライン
├────────────────────────────┤
│  [ド] [レ] [ミ] [ファ]     │  ← タップボタン（固定）
└────────────────────────────┘
```

### Canvas の役割

- 音符（丸や四角）の落下アニメーション（`requestAnimationFrame`）
- 判定エフェクト（PERFECT! / GOOD! のテキスト表示）
- 判定ラインの描画

### タップボタン

- Canvas の下に固定配置（HTML要素）
- `touchstart` / `mousedown` でトリガー（遅延をなくすため）
- `touch-action: manipulation` でダブルタップズーム無効化

## 難易度

- 楽曲を選んだらそのまま開始（難易度選択なし）
- 曲のテンポ自体が難易度（かえるのうたが一番ゆっくりで易しい）
- 将来的に「かんたん / ふつう」モード追加可能（現時点では不要）

## デザイン・UX

- **スマホ縦持ち最優先**（`100dvh`、`safe-area-inset-bottom` 対応）
- **タップターゲット最低 44px**
- フォント：`Hiragino Maru Gothic Pro` > `BIZ UDPGothic`
- カラーテーマ：明るく・カラフル（既存ゲームと統一）
- ヘッダーの「もどる」ボタンで `index.html` に戻る
